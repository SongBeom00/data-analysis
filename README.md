## Apartment & Retail Open-Data Visualizations

이 프로젝트는 공개 데이터셋을 활용해 데이터 시각화 프로젝트 주피터 노트북 모음입니다. 데이터 정제 → 시각화 → 인사이트 도출 과정을 단계별로 기록해, 재현 가능하고 확장 가능한 데이터 시각화 파이프라인을 제시합니다.

### 데이터셋

- `open-data-set/주택도시보증공사_전국 평균 분양가격(2019년 12월).csv`
- `open-data-set/전국 평균 분양가격(2013년 9월부터 2015년 8월까지).csv`
- `상가상권정보/상가업소정보_201912_01.csv`
- `상가상권정보/seoul_municipalities_geo_simple.json`

### 핵심 노트북

| Notebook                        | 설명                                                                                                                    |
| ------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| `00-apt-price.ipynb`            | 2013–2019년 평균 분양가 데이터를 병합하고, 피벗·집계·Seaborn 시각화를 통해 지역/면적별 가격 패턴과 이상치를 분석합니다. |
| `01-store-data-eda-input.ipynb` | 상가 업소·도시공원 데이터를 로드하고, 한글 폰트·레티나 출력 등 시각화 환경을 설정하며 EDA를 준비합니다.                 |

### 개발 환경

1. Python 3.11+ 권장 (프로젝트 루트의 `pyproject.toml` 확인)
2. [uv](https://github.com/astral-sh/uv) 또는 `pip`으로 의존성 설치
   ```bash
   uv sync  # 또는
   pip install -r requirements.txt
   ```
3. VS Code + Jupyter 확장으로 노트북 실행 권장

### 실행 방법

1. 저장소 클론 후 루트(`/01_data_analysis/00_open_data_analysis`)로 이동
2. 가상환경 활성화 및 의존성 설치
3. `jupyter lab` 혹은 VS Code 에서 노트북 실행
4. 셀을 순서대로 실행하며 출력 그래프·표를 확인

### 시각화 하이라이트

- 지역별/전용면적별 평당 분양가 바차트 및 Box/Violin Plot
- 연도·월 단위 히트맵으로 가격 추세 열지도
- FacetGrid, catplot, swarmplot 등으로 다차원 비교
- Seaborn 한글/마이너스 폰트 세팅 및 레티나 해상도 출력

### 향후 계획

- 상가업소/도시공원 데이터와 공간 정보(GeoJSON)를 결합한 지도 시각화
- 이상치 감지 및 가격 예측 모델 추가
- Streamlit 대시보드로 시각화 결과 공유
